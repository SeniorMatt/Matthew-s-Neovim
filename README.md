# Matthew-s-Neovim

## Requirements
You need to have **Omnisharp** installed inside your '~/Documents/omnisharp' folder.

You can do that by going to the official GitHub repo:
https://github.com/OmniSharp/omnisharp-roslyn/releases

And downloading && extracting omnisharp-linux-musl-x64-net6.0.tar.gz where you need.

## Installation
Insert this in to your `~/.config/nvim/`
